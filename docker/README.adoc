Three containers support the Wainlux K6 control system.

== Compose Files

`compose.yaml` runs production mode with real K6 hardware. Needs `/dev/ttyUSB0` access on Pi.

[source,bash]
----
cd docker
docker compose up -d
----

`compose-dev.yaml` runs mock mode without hardware. For testing on Steam Deck or dev machines.

[source,bash]
----
cd docker
docker compose -f compose-dev.yaml up -d
----

Or use deployment scripts: `scripts/deploy_to_pi.sh` (production), `scripts/deploy_to_steamdeck.sh` (dev), `scripts/quick_deploy.sh` (production).
