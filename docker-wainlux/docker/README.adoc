== API

* `GET /` - UI
* `POST /api/connect` - Connect
* `POST /api/disconnect` - Disconnect
* `POST /api/test/home` - Home
* `POST /api/test/bounds` - Test pattern
* `POST /api/engrave` - Engrave image
* `GET /api/status` - Status
